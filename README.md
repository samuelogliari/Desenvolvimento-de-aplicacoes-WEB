# Commits 
## O que é o projeto
  - O meu projeto é basicamente um git para guardar e rever estudos da faculdade voltados para desenvolviemnto web, questões como  progressão de projetos, evolução como programador, entre outros.
---
## Como eu fiz os commits
  - A príncipio eu fiz pelo Github Desktop, e por baixo disso tudo o que ocorreu foi:
  - Começamos com um `git init` para iniciar o repositório;
  - Foi feito o primeiro commit, `git add .`, verifiquei com `git status` e após `git commit -m "Initial commit"`;
  - Enviei os commits por meio de `git push origin -u`, optei usar o `-u` para futuramente usar somente o `git push`;
  - Seguindo a mesma lógica, fui adicionando o conteúdo de outras aulas;
  - Criei um arquivo `index.html` para à pagina principal;
  - Compartilhei com um amigo meu, para testar comandos de `clone` e afins;
